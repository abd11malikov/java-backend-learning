#  Notification System (Polymorphism Demo)

This project is my exploration of **Inheritance** and **Polymorphism** on Day 2 of my Java Backend journey. I built a flexible notification system to understand how to write code that is easy to extend without breaking existing logic.

##  Project Structure
```text
day2-inheritance/
└── src/
    ├── NotificationSender.java   # The Interface
    ├── EmailNotification.java    # Implementation 1
    ├── SmsNotification.java      # Implementation 2
    ├── PushNotification.java     # Implementation 3 
    └── ShowNotification.java     # Testing the polymorphism
